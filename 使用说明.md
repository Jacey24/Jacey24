# GitHub 主页使用说明

1. 在 GitHub 新建一个公开仓库，仓库名必须与你的用户名完全相同：`Jacey24`。
2. 把本文件夹中的 `README.md` 上传到仓库根目录。
3. 把 `assets/cat-banner.svg` 按相同目录结构上传。
4. 提交后访问 `https://github.com/Jacey24` 即可看到新主页。

## 可选调整

- 在 `About me` 下补充你的职业、技术栈、正在学习的内容和联系方式。
- 如果不想展示访问量，删除包含 `komarev.com` 的 `<img>` 标签。
- 如果统计卡片暂时加载失败，通常是公共服务的访问限制，稍后刷新即可。
